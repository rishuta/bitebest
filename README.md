# BiteBest 🍽️

BiteBest is a food price comparison web application developed to help users compare food prices across multiple delivery platforms like Swiggy, Zomato, Magicpin and EatSure.

Users can search restaurants or food items and compare prices, delivery charges, offers and final payable amount to identify the best option available.

This project was developed as a full-stack application using React.js, Node.js, Express.js and MongoDB Atlas.

## Features

- Search food items and restaurants
- Compare food prices across platforms
- View delivery charges and offers
- Best deal recommendation
- Restaurant-wise comparison
- Admin dashboard for managing data
- Mobile responsive interface

## Installation

Clone repository:

```bash
git clone https://github.com/rishuta/bitebest.git
```

Backend:

```bash
cd backend
npm install
npm start
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```
